# Oxford 3000 Word Learning System — Project Instructions

## โปรเจกต์นี้คืออะไร
ระบบเรียนศัพท์ภาษาอังกฤษ Oxford 3000 คำ ใน 365 วัน สำหรับเด็กอายุ 7–12 ปี  
ผลิตคอนเทนต์ด้วย AI ผ่าน Claude.ai Pro + Canva Pro โดยไม่เสีย API เพิ่ม

---

## โครงสร้างระบบ

### การจัดลำดับคำศัพท์
- เรียงตาม Oxford 3000 CEFR level: **A1 → A2 → B1 → B2** ห้ามสุ่ม
- ความยากต้องไต่ระดับเสมอ ทั้งระดับวัน สัปดาห์ และทั้งปี

### รูปแบบรายสัปดาห์
- ~6–8 คำ/วัน จัดเป็น **Theme รายสัปดาห์**
- 1 วัน/สัปดาห์ = วันทบทวน (Review Day)

### 5 ขั้นตอนต่อวัน (ลำดับนี้เสมอ)
1. **ฟัง** — ฟังเรื่องก่อน ยังไม่ดูตัวหนังสือ
2. **อ่าน + ฟัง** — อ่านพร้อมฟังไปด้วย
3. **อ่าน + พูด** — อ่านออกเสียงตาม
4. **ฟัง + เขียน** — Dictation 2–3 ประโยค
5. **แบบฝึกหัด** — เรียงจากง่ายไปยาก

### สิ่งที่แต่ละวันต้องมี
- เรื่องสั้น (original story — ห้ามละเมิดลิขสิทธิ์)
- ไฟล์เสียง 4 เวอร์ชัน: ฟังเฉยๆ / อ่านตาม / พูดตาม / Dictation
- ภาพประกอบ (AI-generated, copyright-free)
- แบบฝึกหัด

## กฎบังคับ (ตรวจทุกครั้งที่ผลิตเนื้อหา)
1. **ลิขสิทธิ์** — ห้ามอิงตัวละคร/พล็อตที่มีเจ้าของลิขสิทธิ์ในทุกเรื่อง ภาพ และเสียง
2. **ลำดับความยาก** — คำศัพท์และเนื้อหาต้องเรียงง่าย → ยากเสมอ
