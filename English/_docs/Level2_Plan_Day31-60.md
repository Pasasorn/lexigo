## กฎเดิมที่ต้องรักษา
1. ลิขสิทธิ์ — ตัวละคร/เรื่อง original เท่านั้น
2. ง่าย → ยาก เสมอ (ในวัน, ในสัปดาห์, ข้ามสัปดาห์)
3. คำในวัน: nouns → verbs → adjectives/adverbs
4. Lock chain: `wla_d31` … `wla_d60` (Day 31 unlock ด้วย code Day 30)
