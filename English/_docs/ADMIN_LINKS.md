## 💡 Admin Tips

```
# เปิด admin mode ใน Browser Console:
localStorage.setItem('wla_admin','1')

# ปิด admin mode:
localStorage.removeItem('wla_admin')

# ดู student info ที่ login อยู่:
JSON.parse(localStorage.getItem('wla_student'))
```
