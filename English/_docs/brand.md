# Brand Identity — Oxford 3000 Word Adventure

## ชื่อแบรนด์
**"Word Adventure"** / **"ผจญภัยคำศัพท์"**  
Tagline: *"Every word is a new adventure!"*

---

## บุคลิกแบรนด์ (Brand Personality)
| มิติ | คำอธิบาย |
|------|----------|
| อารมณ์หลัก | สนุก, อบอุ่น, ให้กำลังใจ, ตื่นเต้น |
| น้ำเสียง | เป็นเพื่อน ไม่ใช่ครู — พูดกับเด็ก ไม่ใช่สั่งเด็ก |
| ค่านิยม | ความพยายามสำคัญกว่าความสมบูรณ์แบบ |
| สิ่งที่ห้ามทำ | ข่มขู่, กดดัน, ใช้ภาษาที่ทำให้รู้สึกโง่ |

---

## Mascot — "Wordy"
- รูปร่าง: หนอนตัวเล็กน้อย สีเขียวมิ้นต์ มีตาโต หูฟังสีส้ม สมุดโน้ตเล็กในมือ
- บุคลิก: ขี้อยากรู้ ร้องเพลงตลอดเวลา ชอบพูดคำศัพท์ใหม่
- ใช้เพื่อ: แนะนำกิจกรรม, ให้กำลังใจ ("You can do it! 🎉"), แสดงผลลัพธ์

---

## จานสีหลัก (Color Palette)

### สีหลัก — ใช้ทุกช่วงอายุ
| สี | Hex | ความหมาย |
|----|-----|----------|
| Sky Blue | `#4FC3F7` | เปิดกว้าง, เรียนรู้ |
| Sunshine Yellow | `#FFD54F` | ความสุข, กระตุ้น |
| Mint Green | `#81C784` | เติบโต, ทำได้ |
| Coral Orange | `#FF8A65` | พลังงาน, สนุก |
| Soft White | `#FAFAFA` | พื้นที่อ่าน, สะอาด |

### ปรับโทนตามช่วงวัย
| ช่วงอายุ | โทนสี | เหตุผล |
|---------|-------|--------|
| 7–8 ปี (A1) | สดใส, contrast สูง, หลายสีพร้อมกัน | ดึงดูดสายตา, สมาธิสั้น |
| 9–10 ปี (A2) | สีหลักชัด + accent 1–2 สี | เริ่มจดจ่อได้นานขึ้น |
| 11–12 ปี (B1–B2) | โทนสะอาด, minimalist มากขึ้น | รู้สึก "โต" และมั่นใจ |

## Typography

| ประเภท | Font แนะนำ | ลักษณะ |
|--------|-----------|--------|
| หัวข้อหลัก | **Fredoka One** | กลมน่ารัก อ่านง่าย |
| เนื้อเรื่อง | **Nunito** | เป็นมิตร ไม่เป็นทางการ |
| คำศัพท์ highlight | **Baloo 2 Bold** | โดดเด่น จำง่าย |
| Thai text | **Sarabun** | อ่านสบาย หน้าจอ/พิมพ์ |

- ขนาดตัวอักษรเนื้อหา: ไม่ต่ำกว่า 16px (จอ) / 14pt (พิมพ์)
- ห้ามใช้ฟอนต์ serif หรือแคบ — เด็กอ่านยาก

---

## ไอคอนและ Visual Elements
- มุมกล่องทุกอัน: **rounded corners** (radius ≥ 12px) — ไม่มีมุมแหลม
- ไอคอนสัญลักษณ์ประจำกิจกรรม:
  - 🎧 ฟัง
  - 📖 อ่าน
  - 🗣️ พูด
  - ✏️ เขียน
  - ⭐ แบบฝึกหัด / ทำได้แล้ว!
- Progress แสดงเป็น: ดาว ❤️ หรือ badge — ไม่ใช้ % หรือคะแนนตัวเลขดิบ

## ระบบให้กำลังใจ (Encouragement System)

### ข้อความกำลังใจตามผลลัพธ์
| สถานการณ์ | ข้อความ |
|---------|--------|
| เริ่มบทเรียน | "Let's go, Word Explorer! 🚀" |
| ทำแบบฝึกหัดถูก | "Amazing! You're a word champion! ⭐" |
| ทำผิด | "Oops! Let's try again — you've got this! 💪" |
| จบบทเรียน | "You learned X new words today! Wordy is so proud! 🎉" |
| ครบสัปดาห์ | "One whole week! You're on fire! 🔥" |

### กฎการให้กำลังใจ
1. **ไม่เคยบอกว่าผิด** — ใช้ "try again" หรือ "almost!"
2. **ชมความพยายาม** ก่อนชมผลลัพธ์เสมอ
3. **ทุก milestone** ต้องมี celebration เล็กๆ (animation, badge, เสียง)

---

## เสียง (Audio Branding)
- BGM: เบา, สนุก, เป็นจังหวะ — แนว children's acoustic / ukulele
- เสียง feedback ถูก: เสียงกระดิ่งสดใส 🔔
- เสียง feedback ผิด: เสียงนุ่มๆ ไม่ตกใจ (เช่น "boing" เบาๆ)
- เสียงพูดคำศัพท์: ชัด, ช้า, สำเนียง American English มาตรฐาน

---

## กฎ DO / DON'T

| DO ✅ | DON'T ❌ |
|------|---------|
| ใช้ภาพสีสด มีชีวิตชีวา | ภาพขาวดำ หรือ stock photo คน |
| ตัวอักษรใหญ่ ชัดเจน | ตัวเล็ก ซีด หรืออ่านยาก |
| Wordy ปรากฏในทุก section | ไม่มี mascot เลย |
| บอกว่าทำได้เสมอ | บอกว่าผิด หรือ "ไม่ถูก" |
| สีตามธีมสัปดาห์ | สีสุ่ม ไม่ coherent |
| มี whitespace เพียงพอ | อัดแน่นจนล้นหน้า |
